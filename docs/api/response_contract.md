# 统一响应格式说明

## 响应结构

所有接口统一返回以下 JSON 结构：

```json
{
  "code": 0,
  "message": "ok",
  "data": {}
}
```

## 字段定义

- `code`：业务状态码
- `message`：业务说明信息
- `data`：返回数据对象（可为空）

## 状态码约定

- `0`：成功
- `40000`：参数错误
- `40100`：未登录
- `40101`：无权限
- `40400`：资源不存在
- `50000`：系统内部错误

## 约束

- 即使发生业务异常，也保持统一结构返回。
- 框架级参数校验异常统一映射为 `40000`。
- 未捕获异常统一映射为 `50000`，避免返回栈信息给客户端。
